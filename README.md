# AI Chat Support Bot with Live Streaming

RAG-based chatbot API: users upload PDFs, ask questions, get streamed answers via WebSockets.

## Stack
FastAPI, PostgreSQL + pgvector, SQLAlchemy/Alembic, JWT auth, AWS (S3/EC2/RDS/IAM), Google Gemini API, Docker.
